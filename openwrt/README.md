## ⚙️ Prerequisites 🛠️

Before you begin your journey, make sure you have the following tools ready:

*   💻 A Linux environment (or WSL) for building images.
*   💾 Necessary build dependencies (GCC, Make, etc.).
*   🧠 A solid understanding of basic networking concepts (IP, routing, VPNs).
*   💡 Access to hardware or virtual machines to test your configurations.
